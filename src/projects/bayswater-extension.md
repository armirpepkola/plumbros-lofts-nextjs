This project in Bayswater involved a modern, single-storey rear extension designed to create a large, open-plan kitchen and dining area. We used large-format sliding glass doors to maximize the connection to the garden, creating a perfect space for entertaining. The exterior brickwork was carefully sourced to seamlessly match the existing Victorian property.
